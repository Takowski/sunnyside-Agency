Welcome !

This is a Sass Boilerplate Project

You can download this from Github and use this as a starter for any Sass Project

What this boilerplate contains

  • Font-Awesome and Animate CSS built in !
  • 7-1 Folder Architecture
  • Global Reset
  • Gitignore included
  • Media Query Manager
  • Javascript Babel Compilation
  • Development scripts : compile, serve & watch
  • Production scripts : compile, prefix and compress
